App này dành cho bao nhiêu người dùng?

Đây là yếu tố quyết định toàn bộ kiến trúc — auth, database, deployment đều phụ thuộc vào đây.

A

Chỉ mình bạn

Tool học tập cá nhân. Không cần đăng nhập phức tạp, SQLite là đủ, deploy đơn giản. Vẫn có server để dùng AI, lưu data tốt hơn localStorage.

✓ Setup nhanh  ·  ✓ Không cần auth  ·  ✓ SQLite / file-based  ·  ✓ 1 server nhỏ là đủ
B

Bạn + vài người nữa (2–20 người)

Chia sẻ với đồng nghiệp/bạn cùng ôn thi PMP. Mỗi người có account riêng, highlight riêng, progress riêng. Vẫn nhỏ, không cần scale lớn.

✓ Auth cơ bản  ·  ✓ PostgreSQL  ·  ✓ Session/JWT  ·  ~ Cần thêm 1 tuần setup
C

Public / Nhiều người (20+ người)

Mở cho cộng đồng hoặc học viên. Cần auth đầy đủ, rate limiting, backup, monitoring. Phức tạp hơn đáng kể.

~ Full auth system  ·  ~ PostgreSQL + Redis  ·  ~ DevOps overhead  ·  ~ 3–4 tuần setup
