Bạn muốn thêm tính năng gì nhờ có server?

Chọn tất cả những gì bạn quan tâm — đây sẽ quyết định tech stack và độ phức tạp.

🤖

AI / Claude integration

Hỏi Claude giải thích khái niệm, tạo câu hỏi thi mới, phân tích điểm yếu của bạn. Cần server để giấu API key.

🔊

Text-to-Speech / Audio

Đọc to nội dung chương để nghe khi lái xe / chạy bộ. Bạn đã có scripts generate-audio.py + piper-setup.sh trong project!

☁️

Sync thực sự

Thay GitHub API hack bằng database thật. Highlights, notes, progress đồng bộ ngay lập tức giữa máy tính và điện thoại.

📝

Quản lý nội dung

Sửa/thêm chương sách qua admin panel thay vì edit HTML thủ công. Serve nội dung từ database hoặc Markdown files.
